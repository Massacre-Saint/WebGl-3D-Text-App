## Run Locally

Clone the project

```bash
  git clone https://github.com/Massacre-Saint/WebGl-3D-Text-App.git
```

Go to the project directory

```bash
  cd 3dtextApp
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
